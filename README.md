# 苗栗南庄銀髮藝術節 | Nanzhuang Senior Arts Festival

這是「觀眾發展與行銷」課程的期末募資專題 Landing Page。

## 專案亮點
- **藝術與情感**：結合長輩圓點藝術與跨世代音樂交響。
- **專業聯名**：法藍瓷 (Franz) 與南藝大應用音樂系合作。
- **策略深度**：包含詳細的 SWOT 2.0 矩陣與市場對標分析。
